ไปหน้าล็อกอิน
Step 1

เลือกตาราง

ลำดับคอลัมน์ที่ระบบจะใช้ (วางเนื้อหาตามลำดับนี้):
ⓘ ตารางนี้สร้าง ID อัตโนมัติ — ใช้โหมด append เท่านั้น
Step 2

วางข้อมูล (เฉพาะเนื้อหา ไม่ต้องเอา header มา)

copy เฉพาะเซลล์ข้อมูล จาก Google Sheets/Excel แล้ว Ctrl+V ที่นี่ — ระบบจะ map คอลัมน์ตามลำดับด้านบน (1, 2, 3, ...)

Step 3

ตรวจ + บันทึก

⚠️ ข้อมูลที่วางมีคอลัมน์เกินจาก schema ( คอลัมน์เกิน) — คอลัมน์ส่วนเกินจะถูกตัดทิ้ง
❗ มี แถวที่ขาดค่า PK () — โหมด upsert ต้องมีทุกแถว / เลือกโหมด append เพื่อให้ระบบสร้าง ID ให้แทน
